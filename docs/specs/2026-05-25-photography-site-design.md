# 个人摄影作品 & 想法网站 — 设计方案

## 项目概述

个人网站，用于展示摄影作品和文章/想法。Apple 式极简美学，极光渐变背景，文件管理内容，免费部署。

## 视觉方向

- **风格**：Apple 极简主义 — 大面积留白、精致排版、克制的 UI
- **背景**：极光渐变效果（深色底 + 流动的蓝/绿/紫光晕）
- **调性**：暗色为主，让照片像画廊展品一样突出
- **动画**：页面切换流畅过渡，微妙的悬停效果

## 网站结构

```
首页 (/)              — 纯入口：极光背景 + 标题 + 导航卡片
├── 摄影 (/gallery)    — 作品集：按系列分组、网格/瀑布流、点击放大
├── 想法 (/thoughts)   — 博客：文章列表 + 阅读页
└── 关于 (/about)      — 个人介绍 + 摄影理念 + 联系方式
```

### 首页
- 全屏极光渐变背景
- 居中：名字/Logo + 一句话简介
- 下方三个导航卡片（摄影 / 想法 / 关于）
- 卡片悬停有微动效

### 摄影作品集
- 按系列/主题分组（如：城市 / 自然 / 人像）
- 照片网格展示（Masonry 或整齐网格）
- 点击单张进入大图浏览（Lightbox）

### 想法/文章
- 文章列表：标题 + 日期 + 摘要
- 文章详情：MDX 渲染，支持图片
- 按时间倒序排列

### 关于我
- 个人照片
- 自我介绍文字
- 摄影理念
- 联系方式 / 社交链接

## 技术方案

| 层级 | 选型 | 理由 |
|------|------|------|
| 框架 | Next.js 14+ (App Router) | React 生态，SSG/SSR 灵活 |
| 样式 | Tailwind CSS | 快速开发，响应式方便 |
| 组件 | shadcn/ui + 21st.dev | 高质量预制组件 |
| 文章 | MDX (next-mdx-remote) | Markdown 写作，支持内嵌组件 |
| 部署 | Vercel | 免费、自动部署、全球 CDN |
| 内容 | 文件管理 | 照片放 /public，文章放 /content |

## 内容管理流程

1. 写文章：在 `/content/thoughts/` 下新建 `.mdx` 文件
2. 传照片：放入 `/public/photos/` 对应系列文件夹
3. 推送 GitHub → Vercel 自动部署

